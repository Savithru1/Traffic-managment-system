Hyderabad Traffic Incident Manager
A Java-based console application for real-time traffic incident logging and prioritization using custom Data Structures.

🚀 Key Features
Priority Queue: Automatically sorts incidents by severity (P1 to P4).

Recent History: Uses a Bounded Stack to track the last 3 reported events.

Persistence: File I/O integration to save/load logs from a .txt file.

Zone Mapping: Coverage for 10 major Hyderabad hubs (HITEC City, Gachibowli, etc.).

🛠️ Tech Stack & DSA
Language: Java

Data Structures: Custom Linked Lists (for Priority Queue and Stack)

Storage: BufferedReader / BufferedWriter (CSV format)
